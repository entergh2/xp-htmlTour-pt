Área de trabalho do Windows

É fácil usar o Windows XP. Primeiro, você verá uma área grande da tela, chamada área de trabalho, e uma área menor, localizada na parte inferior da tela, chamada barra de tarefas. Tudo que é possível fazer no computador aparece dentro de quadros chamados janelas. Você pode abrir quantas janelas desejar ao mesmo tempo — bem como redimensioná-las, movê-las ou empilhá-las novamente em qualquer ordem.
