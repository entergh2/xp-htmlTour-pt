É muito fácil alternar entre vários usuários

Com o Windows XP é ainda mais fácil compartilhar um único computador com vários usuários.

Cada usuário do computador pode criar uma conta independente protegida por senha, com configurações personalizadas e arquivos particulares. Várias contas podem estar ativas no computador ao mesmo tempo. Alternar entre contas diferentes é rápido e simples.

Por exemplo, se alguém deseja verificar o email enquanto você está trabalhando, basta alternar para a conta dessa pessoa. Não é preciso fechar os programas e fazer logoff. O outro usuário não verá seus arquivos enquanto estiver trabalhando; e quando você alternar de volta para a sua conta, não verá os arquivos dele. Você encontrará a tela exatamente como a deixou.
