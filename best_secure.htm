Mais seguro e fcil de gerenciar

O Windows XP Professional apresenta um sistema de arquivos seguro, com criptografia para informaes importantes. As configuraes administrativas relativas segurana garantem que o sistema permanea seguro e reservado.

O Windows XP Professional tambm foi projetado para facilitar o gerenciamento.

Um utilitrio de backup avanado e fcil de usar simplifica a cpia regular de dados para fitas magnticas, discos Zip® e outra mdia de armazenamento.

A Recuperao automatizada do sistema pode at restaurar o sistema operacional em caso de falhas no disco.

O Windows XP Professional pode ser implantado aos poucos em toda a sua organizao. A atualizao de computadores individuais somente quando necessrio reduz os custos e permite que voc se beneficie da nova tecnologia do Windows XP no seu prprio tempo.

Os computadores comerciais conectados pelo software Windows 2000 Server so muito fceis de atualizar e gerenciar usando os servios de instalao remota e as diretivas de grupo. Esses recursos permitem que voc instale, configure e gerencie computadores individuais como grupos em vez de mquinas individuais, e os monitore de uma localizao centralizada com grandes economias de tempo e custos de suporte.
