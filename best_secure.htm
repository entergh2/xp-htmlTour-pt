Mais seguro e fácil de gerenciar

O Windows XP Professional apresenta um sistema de arquivos seguro, com criptografia para informações importantes. As configurações administrativas relativas à segurança garantem que o sistema permaneça seguro e reservado.

O Windows XP Professional também foi projetado para facilitar o gerenciamento.

Um utilitário de backup avançado e fácil de usar simplifica a cópia regular de dados para fitas magnéticas, discos Zip® e outra mídia de armazenamento.

A Recuperação automatizada do sistema pode até restaurar o sistema operacional em caso de falhas no disco.

O Windows XP Professional pode ser implantado aos poucos em toda a sua organização. A atualização de computadores individuais somente quando necessário reduz os custos e permite que você se beneficie da nova tecnologia do Windows XP no seu próprio tempo.

Os computadores comerciais conectados pelo software Windows 2000 Server são muito fáceis de atualizar e gerenciar usando os serviços de instalação remota e as diretivas de grupo. Esses recursos permitem que você instale, configure e gerencie computadores individuais como grupos em vez de máquinas individuais, e os monitore de uma localização centralizada com grandes economias de tempo e custos de suporte.

Zip® é uma marca registrada de Iomega Corporation.
